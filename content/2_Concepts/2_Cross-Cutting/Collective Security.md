# Collective Security

## Definition
A system in which an attack on any member state is treated as an attack on all, and the entire membership collectively responds to restore peace. Unlike [[NATO|collective defence]] (which organises against an external threat), collective security is universal and aims to deter any aggression from any source — including from within the system. The core principle: "all against one" replaces "each for itself."

## Key Institutions
| Institution | Founded | Mechanism | Weakness |
|-------------|---------|-----------|----------|
| **League of Nations** | 1919 (Covenant) | Art. 10-16: collective guarantee, sanctions, military action | No enforcement power; unanimity voting; US never joined |
| **United Nations** | 1945 (Charter) | Ch. VII: Security Council authorises force, peacekeeping | P5 veto paralyses action where permanent member interests are involved |

## Core Assumptions of Collective Security
1. **All members agree** that any aggression harms the entire system
2. **All members will contribute** to stopping aggression, regardless of their own interests
3. **Enough power** exists collectively to deter or defeat any aggressor
4. **The system is impartial** — it enforces rules, not great power interests

## Collective Security vs Collective Defence
| Dimension | Collective Security | Collective Defence |
|-----------|-------------------|-------------------|
| Target | Any aggressor (including member) | External aggressor (typically identified) |
| Membership | Universal or near-universal | Closed group with common interests |
| Rationale | System maintenance | Self-preservation, alliance solidarity |
| Theory | [[Liberalism]], idealism | [[Realism]] |
| Example | UN Charter, League Covenant | [[NATO]] Art. 5 |
| Mechanism | All against one | Alliance against designated threat |

## Key Debates
- **Feasibility** — Realists argue collective security violates the fundamental logic of national interest. States will not sacrifice for an abstract system
- **The P5 problem** — The UN system built collective security on great power consensus (P5 veto). When P5 disagrees, the system stalls (Syria, Ukraine)
- **Selective enforcement** — Kosovo (1999) without UN mandate; Iraq (2003) without UN approval. "Collective" becomes political
- **Article 43 problem** — UN Charter contemplated standing forces under UN command (Art. 43). Never implemented — forces remain national
- **Peacekeeping invention** — When Ch. VII enforcement proved impossible, peacekeeping (Ch. VI ½) emerged as a weaker alternative

## Key Questions
- Can collective security work without a hegemon?
- Is the UN system collective security in practice, or collective defence of the post-1945 order?
- Does the [[Responsibility to Protect]] revive or undermine collective security?
- Is collective security always preferable to alliance security?

## Case Studies
- **League of Nations failures** — Manchuria (1931): Japan invades; League condemns but no action. Abyssinia (1935): Italy invades; sanctions fail, Hoare-Laval Pact discredits League
- **Korean War (1950)** — Soviet boycott of Security Council enabled UN-authorised force. Only example of full Ch. VII enforcement against a state
- **Gulf War (1991)** — Iraq invades Kuwait; UN authorises force (Res. 678). Coalition succeeds. Seen as collective security functioning as designed
- **Syria (2011–present)** — P5 deadlock (Russia vetoes resolutions). Collective security fails to respond to mass atrocity
- **Ukraine (2022)** — Russia vetoes its own condemnation. General Assembly bypass via Uniting for Peace resolution (non-binding)

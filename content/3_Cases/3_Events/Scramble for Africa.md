# Scramble for Africa / Berlin Conference (1884–85)

## Overview
European powers partitioned Africa without any African representation at the Berlin Conference. The central case study for postcolonial IR and the colonial origins of the modern state system.

## Key Themes
- Colonialism and imperialism
- Artificial borders and postcolonial state weakness
- Sovereignty as European construct imposed globally
- Extraction, exploitation, underdevelopment
- Westphalian sovereignty exported by force

## Related Theories
- [[Postcolonialism]] — foundational case: how Europe created the "Third World"
- [[Marxism]] — primitive accumulation, colonial extraction
- [[Realism]] — European balance of power extended to Africa
- [[Sovereignty]] — Westphalian sovereignty imposed on non-European societies
- [[International Development]] — underdevelopment as legacy

## Key Questions
1. How did the Scramble create the conditions for postcolonial conflict?
2. What is the relationship between colonialism and modern IR?
3. How does postcolonial theory reframe this event?
4. Did colonialism create or destroy sovereignty?

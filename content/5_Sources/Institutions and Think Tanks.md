## US Think Tanks

| Institution | URL | Description |
|---|---|---|
| **Council on Foreign Relations (CFR)** | [cfr.org](https://www.cfr.org) | Independent, nonpartisan. Founded 1921. Flagship: *Foreign Affairs* journal. |
| **Brookings Institution** | [brookings.edu](https://www.brookings.edu) | Oldest US liberal/centrist think tank. Governance, economics, foreign policy. |
| **Carnegie Endowment for International Peace** | [carnegieendowment.org](https://carnegieendowment.org) | Global offices (Wash., Moscow, Beijing, Beirut, New Delhi). Nuclear Policy Program. |
| **Center for Strategic and International Studies (CSIS)** | [csis.org](https://www.csis.org) | Bipartisan policy research. *Critical Questions*; *Strategic Asia* series. |
| **RAND Corporation** | [rand.org](https://www.rand.org) | Rigorous quantitative analysis. Defense, security, public policy. |
| **Stimson Center** | [stimson.org](https://www.stimson.org) | International security, shared prosperity. *38 North* (North Korea). |
| **Woodrow Wilson Center** | [wilsoncenter.org](https://www.wilsoncenter.org) | Congressional-chartered. Cold War International History Project. |
| **Hudson Institute** | [hudson.org](https://www.hudson.org) | Conservative. Herman Kahn legacy. Security, economics, energy. |
| **Heritage Foundation** | [heritage.org](https://www.heritage.org) | Conservative. *Index of Economic Freedom*. |
| **American Enterprise Institute (AEI)** | [aei.org](https://www.aei.org) | Conservative-leaning. Enterprise, defense, foreign policy. |
| **Center for a New American Security (CNAS)** | [cnas.org](https://www.cnas.org) | Bipartisan. Tech-security nexus. |
| **Quincy Institute for Responsible Statecraft** | [quincyinst.org](https://quincyinst.org) | Transpartisan restraint. *Responsible Statecraft* magazine. Est. 2019. |
